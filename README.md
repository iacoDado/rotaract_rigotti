# Rotaract Trento

![Version](https://img.shields.io/badge/version-1.0-blue)
![License](https://img.shields.io/badge/license-Apache%202.0-green)

Sviluppo di un sito rappresentativo per l’associazione no profit **Rotaract**, in particolare per la divisione relativa a **Trento**.

## Installazione

Nel terminale bash di Visual Studio Code usare il comando, nella cartella che si vuole adibire al progetto:

```bash
git clone [https://github.com/ScarpeDiGomma/rotaract_rigotti.git](https://github.com/ScarpeDiGomma/rotaract_rigotti.git)
```

## Uso
Usare la funzione di run di Visual Studio Code per visualizzare il sito
